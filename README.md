# Morocco With You - Smart Travel Platform

Monorepo Turborepo pour la plateforme SmartTravel.

## Structure

- apps/web: frontend public (Next.js)
- apps/admin: back-office admin (Next.js)
- apps/api: API backend (Fastify)
- packages/db: Prisma schema and database client
- packages/shared: types, validation, utilities shared across apps

## Roadmap MVP

- Catalogue experiences and providers
- Booking flow and payment integration
- Travel planning itineraries
- Group trips module
- Admin dashboard and commission reporting
